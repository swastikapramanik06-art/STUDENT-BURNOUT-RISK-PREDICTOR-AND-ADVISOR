# Student Burnout Risk Predictor and Advisor

## 📌 Project Overview

This is a full-stack web application built using Django and Machine Learning that predicts student burnout risk levels and provides personalized suggestions.

## ⚙️ Tech Stack

* Python
* Django
* Scikit-learn
* HTML, CSS

## 🚀 Features

* User login & registration
* Burnout prediction system
* Dashboard with results
* Personalized advice

## 🛠️ Setup Instructions

```bash
git clone https://github.com/your-username/STUDENT-BURNOUT-RISK-PREDICTOR-AND-ADVISOR.git
cd STUDENT-BURNOUT-RISK-PREDICTOR-AND-ADVISOR
pip install -r requirements.txt
python manage.py runserver
```
